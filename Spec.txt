.NET Core

Компоненты:

1) AuthService - сервис авторизации пользователей
  своя база данных SQLite
  технологии: WebApi, EntityFramework
  - регистрация нового пользователя
  - авторизация
  - проверка доступа
  - получение данных о пользователе

2) PhotoService - сервис фотографий
  своя база данных SQLite
  технологии: WebApi, EntityFramework
  - добавление, удаление фотографий
  - просмотр фотографий
  - создание, удаление альбомов фотографий

3) MainService - главный сервис; раскидывает сообщения остальным сервисам
  технологии: WebApi
  для каждого сервиса своя очередь

4) WebAppVue - веб приложение для пользователя
  технологии: Vue

- Сервисы общаются друг с другом через QueueService
- Сервисы знают только о своих очередях с сообщениями
- Если сервис А хочет обратиться к сервису Б, то он посылает сообщение в QueueService, QueueService знает в какую очередь нужно положить это сообщение, чтобы его принял сервис Б, ответ от сервиса Б приходит снова в QueueService и QueueService клает ответ в очередь сервиса А
